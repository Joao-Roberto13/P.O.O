### Gerenciamento de Sócios###
--------------------------------------------------------------
Este projeto é um sistema de gerenciamento de sócios desenvolvido para auxiliar,
 no controle de informações e atividades relacionadas a uma organização de sócios(clientes).

## Executando o Programa##
--------------------------------------------------------------------------------------
Para executar o programa, siga as instruções abaixo:
1. Certifique-se de ter o ambiente de desenvolvimento configurado corretamente, 
incluindo a versão apropriada do compilador e quaisquer dependências necessárias(SwingX e JCalendar).
2. Navegue até a pasta "src" do projeto.
3. Execute o executável fornecido.

## Estrutura do Projeto ##
A estrutura do projeto é organizada da seguinte maneira:
- **src**: Contém os arquivos fonte do projeto.
  - **console**: Contém os arquivos relacionados ao trabalho sem interface gráfica.
  - **gui**: Contém os arquivos relacionados ao trabalho com interface gráfica.
- **docs**: Documentação relacionada ao projeto, se aplicável.
- **test**: Testes unitários ou arquivos de teste, se houver.
